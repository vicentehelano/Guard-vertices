# Guarded-vertices
*A guard-based data structure for triangulations*

This repository is under development.
It will contain a Python implementation of a compact data structure for triangulations using the concept of vertex-guards (Batista, 2010).
At the moment, we have only an uncompressed implementation of Blandford et al. (2005) data structure.

## References

1. Batista, V. H. F., [Transversais de triângulos e suas aplicações em triangulações](http://objdig.ufrj.br/60/teses/coppe_d/VicenteHelanoFeitosaBatista.pdf). PhD thesis, Universidade Federal do Rio de Janeiro, COPPE, Civil Engineering Program, 2010.
2. Blandford, D. K. et al., [Compact representations of simplicial meshes in two and three dimensions](https://www.worldscientific.com/doi/abs/10.1142/S0218195905001580). International Journal of Computational Geometry \& Applications, v. 15, n. 1, p. 3-24, 2005.
